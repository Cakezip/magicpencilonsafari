Magic Pencil Help

Quick answers for setup, troubleshooting, and release support preparation.

Extension Does Not Appear

  1. Make sure the app is installed on the iPad.
  2. Open Settings > Apps > Safari > Extensions.
  3. Enable Magic Pencil.
  4. Reload the current Safari page once.

Pencil Still Scrolls

Update to the latest build, reopen Safari, and test again on a standard website. If the issue remains, capture the page URL, iPadOS version, and Safari version for debugging.

The Overlay Does Not Draw

Test on a normal http or https website. Some Apple system pages, PDF views, or restricted pages may not behave like standard web content.

Strokes Disappear Too Soon

Open the Safari extension popup and switch from Volatile mode to Session mode if you want notes to remain until the page reloads, changes, or Safari closes.

Privacy Questions

Open the in-app privacy policy for a summary of on-device storage and the app’s no-account, no-analytics design.
