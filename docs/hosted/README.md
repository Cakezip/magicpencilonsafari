# Hosted Pages

These files are ready to publish on a simple static host such as:
- GitHub Pages
- Cloudflare Pages
- Netlify
- Vercel static hosting

Included pages:
- `index.html`
- `privacy-policy.html`
- `support.html`

GitHub Pages is already wired for this repo through:
- `.github/workflows/deploy-pages.yml`

Recommended App Store Connect URLs:
- Support URL: `https://<your-pages-domain>/support.html`
- Privacy Policy URL: `https://<your-pages-domain>/privacy-policy.html`

If you use GitHub Pages:
1. Push this repository to GitHub.
2. Open `Settings > Pages`.
3. Set `Source` to `GitHub Actions`.
4. Push to `main` and wait for the `Deploy Hosted Pages` workflow to finish.
