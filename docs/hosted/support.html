Magic Pencil Support

Magic Pencil adds a temporary Apple Pencil overlay to Safari pages on iPad.

Setup

  1. Install the app on iPad.
  2. Open Settings > Apps > Safari > Extensions.
  3. Enable Magic Pencil.
  4. Reload the Safari page once if needed.

Troubleshooting

  • If the extension does not appear, confirm the app is installed and the extension is enabled.
  • If Pencil input still scrolls, update to the latest build and retest on a normal website.
  • If notes disappear too soon, switch from Volatile mode to Session mode in the Safari extension popup.
  • Some system pages and nonstandard content views may not behave like normal web pages.
